IDEインストール

  1. mruby/c IDEをダウンロード後、解凍
  2. mruby/c IDEの解凍したフォルダのmrubyc_ide.exeを起動
  3. 上部タブの「File -> Settings...」をクリック
  4. 「Build and Deploy -> Cmpiler」の「mrbc」項目に解凍したフォルダ内にある「mrbc.exe」のpathを指定
  5. 「Build and Deploy -> Writer」の「mrbwrite」項目に解凍したフォルダ内にある「mrbwrite.exe」のpathを指定
  6. 「Build and Deploy -> Writer」の「Options」項目に「-s 19200」と書き込み、「Apply」、「OK」をクリック
