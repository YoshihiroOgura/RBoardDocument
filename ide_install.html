IDEインストール

  • mruby/c IDEをダウンロード後、解凍
  • mruby/c IDEの解凍したフォルダのmrubyc_ide.exeを起動
  • 上部タブの「File -> Settings...」をクリック
  • 「Build and Deploy -> Cmpiler」の「mrbc」項目にmrubyコンパイラ1.4を解凍したフォルダ内にある「mrbc.exe」のpathを指定
  • 「Build and Deploy -> Writer」の「mrbwrite」項目にmruby/c IDEを解凍したフォルダ内にある「mrbwrite.exe」のpathを指定
  • 「Build and Deploy -> Writer」の「Options」項目に「-s 19200」と書き込み、「Apply」、「OK」をクリック
