サーミスタ温度センサ

温度センサ：http://wiki.seeedstudio.com/Grove-Temperature_Sensor_V1.2/

adc = ADC.new(20)
temp = 0
while true
    a = adc.read_v
    a = (3.3 / a)-1
    temp = 1.0/(Math.log(a)/4275+1/298.15)-273.15
    sleep(1)
    puts("ans:" + temp.to_s)
end

ボリューム制御

ボリュームを使用したLEDの明るさ制御
LED:https://www.switch-science.com/catalog/1251/
ボリューム：https://www.switch-science.com/catalog/805/

adc = ADC.new(20)
pwm = PWM.new(16)
pwm.period_us(100)
while true
    a = adc.read_v
    # adc値 × 100％ ÷ adcの最大値
    a = (a*100/3.3).to_i
    pwm.duty_us(a)
end
