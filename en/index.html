RBoard with PIC32 & mruby/c

Introduction

Welcome to the world of mruby/c
mruby/c is a very compact language for microcontrollers compared to CRuby and mruby.

RBoard is an introductory board for those who want to use Ruby (mruby/c) on microcontrollers.
It is implemented as low-level as possible and can support various sensors.

Since mruby/c is installed as standard, development using mruby/cIDE is possible without using a dedicated IDE such as MPLABX.

In addition, since it is equipped with a GROVE port, various sensors and actuators can be used without soldering.

Information about each version can be found on the release page=> Release Page
