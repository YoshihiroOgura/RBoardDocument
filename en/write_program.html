RBoard

言語: 🇯🇵 日本語 / 🇺🇸 English
Execute IDE install as a preparation.

Windows10

  1. Click "File -> New -> Project".
  2. Enter the project name and click "OK". (sample project name :『test』)
  3. After right-clicking the created project folder, click "Add New...".
  4. After entering the name of the Ruby file to add, click "OK". (sample file name :『test.rb』)
  5. Connect the board and PC using a USB cable.
  6. After writing the program, click the "Write" button.（Start connection....）
  7. Press the reset switch on the board.(labeled [RS])
  8. 「The process "Write" exited normally.」is displayed, it is completed.
※If the program doesn't burn, something else is using the COM port. (terminal software, etc.)
Try reinserting the USB.
Alternatively, the COM port to write to may be different.(Check the port in "Build and Deploy -> Writer".)。
