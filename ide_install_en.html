IDE install

language：English/日本語

Windows10

  1. Download the mruby/c IDE. extract from zip file.
  2. Start mrubyc_ide.exe in the unzipped folder.
  3. Click "File -> Settings..." on the menu bar.
  4. Specify the file path of "mrbc.exe" in the unzipped folder to the "mrbc" item of "Build and Deploy -> Cmpiler".
  5. Specify the path of "mrbwrite.exe" in the unzipped folder in the "mrbwrite" item of "Build and Deploy -> Writer".
  6. Enter "-s 19200" in the "Options" item of "Build and Deploy -> Writer".
  7. Click Apply and OK.
