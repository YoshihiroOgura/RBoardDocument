GPIO,その他サンプル

Lチカ(オンボードLED)


while true
  leds_write(1)
  sleep(1)
  leds_write(0)
  sleep(1)
end
			

led = GPIO.new(0)
while true
  led.write(1)
  sleep 1
  led.write(0)
  sleep 1
end

Lチカ(外部LED)

20番ピンにLEDを接続した場合。 

led = GPIO.new(20)
while true
  led.write(1)
  sleep 1
  led.write(0)
  sleep 1
end

スイッチ入力(オンボードスイッチ)


while true
  status = sw()
  leds_write(status)
end

外部入力(DigitalGROVE)

http://wiki.seeedstudio.com/Grove-Button/ 

btn = GPIO.new(16)
btn.setmode(1)
while true
  status = btn.read()
  leds_write(status)
end

7セグメントLED

74HC4511を使用するとleds_write(0,1,5,6pin)から簡単に出力できる。 

j = 0
while true
  leds_write(j)
  j = j + 1
  if(j==10)
    j = 0
  end
  sleep(1)
end
